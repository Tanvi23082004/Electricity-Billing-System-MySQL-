# Electricity-Billing-System-MySQL-
Electricity Billing System
Project Overview

The Electricity Billing System is a database management project developed using MySQL to automate and streamline electricity billing operations. The system efficiently manages consumer records, electricity connections, meter details, meter readings, bill generation, and payment tracking through a structured relational database.

Objectives
Maintain consumer and connection records.
Record and manage meter readings.
Calculate electricity consumption.
Generate electricity bills automatically.
Track payment status and payment methods.
Perform data retrieval and analysis using SQL queries.
Features
Consumer Management

Relational Database Design
Database Tables

The project consists of the following tables:

Consumers
Connection
Meter
Meter_Readings
Bills
Payments

These tables are linked using Primary Keys and Foreign Keys to maintain data integrity and relationships.
